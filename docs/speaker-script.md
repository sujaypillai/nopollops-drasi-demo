# Speaker script

## Hook

"Every platform team has polling scripts. They ask Kubernetes, databases, registries, and APIs the same question again and again: did anything change?"

## Drasi explanation

"Drasi flips that model. Sources watch systems of record. Continuous queries declare the change that matters. Reactions receive precise additions, updates, and deletes from the query result."

## Demo narration

"The audience is now a source of change. Their app submissions land in PostgreSQL. The workloads run in Kubernetes. Drasi joins both sources and keeps the answer current."

## Close

"Don't poll for platform truth. Declare the truth you care about and react when it changes."
